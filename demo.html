Webflow Integration Instructions

1. Add Component Files

Upload the component files to your Webflow project or host them externally:

<link rel="stylesheet" href="dist/index.css">
<script src="dist/index.js"></script>

2. Add Container Elements

Add these HTML elements where you want the components to appear:

// Main careers listings
<div id="fm-careers-webflow-root"></div>

// Featured job listings
<div id="fm-careers-webflow-featured"></div>

Webflow-Specific Features

  • CMS Integration: Works seamlessly with Webflow CMS collections
  • Responsive Design: Automatically adapts to Webflow's responsive breakpoints
  • Custom Styling: CSS classes that can be customized in Webflow Designer
  • No Server Required: Pure client-side component that works with static hosting
  • SEO Friendly: Server-side rendering compatible
  • Performance Optimized: Lightweight and fast loading

Features

  • Three Component Types:
    • Main Listings: Full job listings with filtering and pagination
    • Featured Listings: Priority/featured job positions
    • Application Form: Direct job application integration
  • Job Listings: Display all available job positions from Greenhouse API
  • Filtering: Filter jobs by location and department
  • Pagination: Navigate through job listings with pagination
  • Responsive Design: Mobile-friendly interface
  • Dark Mode Support: Automatic dark mode detection
  • Standalone: No WordPress or PHP dependencies required
